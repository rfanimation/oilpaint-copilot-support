# OilPaint Copilot

**Your studio companion for oil painting**

OilPaint Copilot is the essential iOS app for oil painters who want to improve their craft with science-backed color mixing, smart layer tracking, and intelligent workflow guidance.

---

## Features

### Precise Color Mixing
Get accurate paint mixing recipes using Kubelka-Munk subtractive color theory. Select any target color and see exactly which paints to mix—with real-world quantities like "pea-sized" and "dime-sized."

### Layer Tracking & Fat-Over-Lean
Record each painting layer with the paints and mediums you use. The app tracks fat-over-lean ratios automatically and warns you before mistakes happen.

### Drying Time Alerts
Never rush a painting again. Get notified when layers are dry based on paint type, medium choice, and application thickness.

### 728 Paints from 8 Major Brands
Access a comprehensive database including Gamblin, Williamsburg, Winsor & Newton, Old Holland, Sennelier, Michael Harding, Rembrandt, and Utrecht.

### Reference Image Color Extraction
Import reference photos and extract colors directly. Build your palette from the colors you actually see.

### Limited Palette Mode
Work with classic limited palettes like Zorn, Velázquez, and Split Primary. Learn to mix any color from just a few paints.

### Palette Analysis
Analyze your paint collection for gaps and get recommendations. Build a complete, archival-quality palette over time.
